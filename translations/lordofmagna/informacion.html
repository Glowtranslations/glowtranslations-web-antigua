¡Lord of Magna: Maiden Heaven en español!
Habiendo comenzado ya con la traducción de Rune Factory 4, nos ponemos manos a la obra con otro juego, que es más corto, pero igual de entretenido:
Lord of Magna: Maiden Heaven, otro titulazo de la mano de Yoshifumi Hashimoto (otros títulos suyos son Rune Factory 4 y Story of seasons).

¡Ya lo tenemos todo listo para empezar! Eso sí, le dedicaremos menos tiempo que al Rune Factory 4, ya que es un proyecto secundario.

¿Cómo aplico el parche?

Hay 4 métodos para parchear el juego: Luma, NTR, 3DS y CIA.
Dependiendo de cuál elijas, tendrás que hacer un procedimiento u otro.
Toda la información que necesitáis se encuentra en el PDF que contiene el parche.

He encontrado un error, ¿cómo puedo reportarlo?

Hemos creado un repositorio en nuestro github para este tipo de cosas y más.
El repositorio de github se encuentra aquí.
Sigue las instrucciones del readme que hay para poder reportar el error.

El parcheador no funciona, ¿qué pasa?

Hemos añadido un nuevo apartado en el Github de Lord Of Magna Maiden Heaven para este tipo de casos. Se encuentra aquí.
Si encuentras un error que no aparece en el F.A.Q., avísanos desde el apartado de "Issues" y te respondemos lo antes posible. El juego lo puedes adquirir de manera digital o en físico.
  • Amazon USA (Físico)
  • Nintendo eshop EUR (Digital)
  • Nintendo eshop USA (Digital)
